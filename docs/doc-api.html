Get record from NerualChain by ID

request:

[GET] http://localhost:9902/record/{record ID}

Example:

http://localhost:9902/record/2844

Get balance a wallet

request:

[GET] http://localhost:9902/[account number]/balance/
Example: 
[GET] http://localhost:9902/be26c9efd14ccd1348d0a8da61c0fa3d8fba095e/balance/
response: 
{
  cm: "balance",
  acc_num: "be26c9efd14ccd1348d0a8da61c0fa3d8fba095e",
  balance: 20
}
